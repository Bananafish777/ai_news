# 前端 UI 设计规范（UI Guidelines）

## 1. 设计系统
- 采用现代简洁风格，强调信息层级、留白和可读性。
- 支持深色/浅色主题切换。
- 主要色彩建议：
  - 主色：#2563eb（蓝）
  - 辅助色：#f59e42（橙）、#10b981（绿）、#f43f5e（红）
  - 灰阶：#f3f4f6、#e5e7eb、#6b7280、#111827
- 字体：系统字体优先，中文推荐“思源黑体”，英文推荐“Inter”。
- 圆角建议：按钮/卡片 8px，输入框 6px。
- 阴影：适度卡片阴影，hover 时增强。

## 2. 组件库
- 推荐使用 Ant Design、MUI 或自研组件库，需二次封装统一风格。
- 主要组件：
  - 按钮（Button）
  - 输入框（Input/TextArea）
  - 表单（Form）
  - 列表（List/Table）
  - 卡片（Card）
  - 标签（Tag/Chip）
  - 弹窗（Modal/Dialog）
  - 通知（Notification/Message）
  - 分页（Pagination）
  - 加载（Spinner/Skeleton）
  - 导航栏（NavBar/SideMenu）
  - 面包屑（Breadcrumb）
  - Tabs、Collapse、Tooltip、Popover 等
- AI 相关：对话气泡、摘要卡片、推荐滑块等自定义组件

## 3. 命名规范
- 采用 BEM 或驼峰式命名（如 news-card、ArticleList、ai-summary-card）。
- 组件文件夹与文件名统一（如 /components/ArticleCard/ArticleCard.tsx）。
- 状态变量、props、API 字段与接口文档保持一致。
- CSS 变量统一前缀（如 --ai-primary-color）。

## 4. 布局规范
- 响应式设计，优先移动端适配（Mobile First）。
- 栅格系统：12 列，常用断点 375/768/1200px。
- 顶部导航 + 侧边栏（管理后台/大屏）+ 主内容区。
- 卡片式内容展示，留白充足，分区明显。
- 列表页：顶部筛选/搜索，内容区分页。
- 详情页：主内容左，侧栏展示推荐/相关/操作。
- 弹窗/抽屉用于编辑、AI 交互等临时操作。

## 5. 交互规范
  - 问答区支持上下文多轮对话，滚动到底部。
  - 摘要/推荐生成有 loading 骨架，失败可重试。

---
### 6.1 顶部导航栏（NavBar）
- 固定顶部，包含网站Logo、主要导航菜单（首页、主题、AI、登录/用户头像下拉等）。
- 响应式收缩为汉堡菜单。
- 支持主题切换按钮。

### 6.2 文章卡片（ArticleCard）
- 展示文章标题、摘要、来源、作者、时间、封面图。
- 支持点击跳转详情页。
- 卡片右上角可有收藏按钮。
- 悬浮有阴影增强。

### 6.3 文章列表（ArticleList）
- 由 ArticleCard 组成，支持分页。
- 顶部有筛选（来源、主题、关键词）、排序（最新/最热）。

### 6.4 文章详情（ArticleDetail）
- 标题、作者、时间、来源、正文（支持富文本/Markdown）、媒体（图片/视频）。
- 右侧/底部展示推荐文章、AI 摘要、AI 问答入口。
- 支持收藏、分享、主题标签。

### 6.5 主题标签（TopicTag）
- 用于文章、主题页，颜色区分，支持点击筛选。

### 6.6 搜索框（SearchBar）
- 首页、列表页顶部，支持输入关键词、回车/按钮搜索。
- 支持自动补全建议。

### 6.7 分页器（Pagination）
- 简洁明了，支持跳页、每页数量切换。

### 6.8 AI 问答区（AIChatBox）
- 气泡式对话，支持多轮上下文。
- 输入区支持快捷键发送、loading、失败重试。

### 6.9 AI 摘要卡片（AISummaryCard）
- 展示 Gemini 生成的摘要，loading 骨架，失败重试。

### 6.10 推荐滑块（AIRecommendSlider）
- 横向滑动展示推荐文章卡片。

### 6.11 登录/注册表单（AuthForm）
- 支持邮箱/用户名、密码输入，校验、错误提示。
- 登录后显示用户头像及下拉菜单。

### 6.12 通用按钮（Button）
- 主按钮、次按钮、危险按钮、禁用/加载态。

### 6.13 通知/消息（Notification）
- 顶部/右上角弹出，自动消失，支持成功/失败/警告类型。

### 6.14 加载与骨架屏（Loading/Skeleton）
- 页面、卡片、AI 结果等加载时统一风格。

### 6.15 管理后台表格（AdminTable）
- 支持排序、筛选、批量操作、分页。
